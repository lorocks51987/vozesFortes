# Instituto Vozes Fortes - Landing Page

Este repositório contém a landing page do Instituto Vozes Fortes construída com Vite, React, TypeScript, Tailwind CSS e componentes shadcn.

## Requisitos

- Node.js e npm

## Desenvolvimento

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Tecnologias

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui
